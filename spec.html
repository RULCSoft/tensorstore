TensorStore JSON Spec¶

TensorStore relies on a JSON Spec format for specifying all parameters necessary for opening or creating a TensorStore.

TensorStore Spec JSON Schema¶

Common properties supported by all TensorStore drivers. Refer to the driver documentation for the supported driver identifiers and driver-specific properties.

object with members:

driver
Required
string
Driver identifier

Specifies the TensorStore driver.
context
Context

Specifies context resources that augment/override the parent context.
dtype
Data type

Specifies the data type.
rank
integer [0, ∞)
transform
Index transform

Specifies a transform.

Data type JSON Schema¶

“bool” | “char” | “byte” | “int8” | “uint8” | “int16” | “uint16” | “int32” | “uint32” | “int64” | “uint64” | “float16” | “float32” | “float64” | “complex64” | “complex128” | “string” | “ustring” | “json”
